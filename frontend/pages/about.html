API Reference
POST
/chat
Send a user message and receive an AI response with optional source citations.
POST
/ingest/upload
Upload a PDF or document file for chunking, embedding, and Qdrant indexing.
POST
/ingest/github
Index a GitHub repository by URL. Extracts README, description, and topics.
POST
/ingest/linkedin
Sync a LinkedIn profile URL or parse an uploaded data export ZIP.
GET
/metrics/langsmith
Returns recent run traces with latency, token usage, and cost data from LangSmith.
GET
/metrics/ragas
Returns RAGAS eval scores (faithfulness, answer_relevancy, context_recall) from local store.
